# Amazon Security Lake (amazon-security-lake)
Amazon Security Lake is a service that automatically centralizes an organization's security data from cloud, on-premises, and custom sources into a purpose-built data lake. It manages the data lifecycle to help you optimize storage and supports OCSF (Open Cybersecurity Schema Framework).

## APIs

### Amazon Security Lake API
The Amazon Security Lake API provides programmatic access to create and manage data lakes, data sources, subscribers, and log sources for centralizing and analyzing security data across your organization.
